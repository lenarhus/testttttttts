preparation

you will need to get 3 accounts/api keys

  • a telegram bot token
  • an account on heroku
  • an account on blocktrail

telegram bot token

Message @botfather https://telegram.me/botfather with the following text: /newbot

@botfather replies with “Alright, a new bot. How are we going to call it? Please choose a name for your bot.”

Type whatever name you want for your bot.

@botfather replies with “Good. Now let’s choose a username for your bot. It must end in bot. Like this, for example: TetrisBot or tetris_bot.”

Type whatever username you want for your bot, minimum 5 characters, and it must end with bot. For example: whatever_bot

@botfather replies with:

“Done! Congratulations on your new bot. You will find it at telegram.me/whatever_bot. You can now add a description, about section and profile picture for your bot, see /help for a list of commands.

Use this token to access the HTTP API: 123456789:AAG90e14-0f8-40183D-18491dDE“

this is the token you will need for later.

heroku account

go to https://signup.heroku.com/ and create an account

blocktrail account

go to https://www.blocktrail.com/dev/signup and create an account

now go to https://www.blocktrail.com/user/settings/apikeys and create a new key, you need both the API Key and the API Key Secret

finally head over to https://www.blocktrail.com/user/BTC/wallets and create either a new bitcoin wallet or a testnet wallet, you will need the wallet name and wallet password

setup

you will need to install the heroku command line interface

windows: 64-bit / 32-bit

osx: brew install heroku

now type heroku login and enter your heroku account information

make sure you have git installed for this next step (https://git-scm.com/downloads)

now go to the folder where you have extracted the supplied source code (either by cding to the directory or in windows shift-right-click in explorer, open command line here)

type heroku create, and write down the randomly generated name

initialize the git repo with git init and then type heroku git:remote -a [your app name]

we need to setup a database, type heroku addons:create heroku-postgresql:hobby-dev

configuration

open up index.js and find the options under // CONFIGURE THESE SETTINGS
set the variable TESTNET to true if you want to test the bot on the test bitcoin network

inside of blocktrail_opts, replace apiKey and apiSecret with your blocktrail api keys
replace WALLET_NAME and WALLET_PASS with your wallet name and password

TELE_TOKEN is where your full telegram token should go, open up find_user_ids.js and replace it there as well

you will need to insert whatever user ids you want into ADMIN_LIST, APPROVAL_ADMIN_LIST, and DEV_ADMIN_LIST, if you don’t know the user ids see the next section..

upload code

make sure you save your changes to index.js and find_user_ids.js

now to upload the code type the following 3 commands (use these any time you wish to upload code to the server)

git add .
git commit -m "[your commit message]"
git push heroku master

once the server has finished installing everything, if you need to find out user ids type
heroku run node find_user_ids.js

have your admins message the bot (not just typing /start) and it will respond with their user_ids, you want to copy these numbers inside the corresponding ADMIN_LIST variables separated by commas (,). once you are done finding user ids, press Ctrl-C in the terminal to stop running the find_user_ids script.

save index.js and commit the code again (git add... git commit... git push)

now to setup the database type heroku run node db_setup.js, if you ever mess up the database replace db_setup with db_delete and then run db_setup again, you should see successfuly created db in the console.

run

you are now ready to run the bot, type heroku ps:scale web=1 and it should work

any time you commit code the bot will restart

if you want to read the bot logs type heroku logs --tail

happy hacking
